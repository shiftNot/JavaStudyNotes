# 1． 训练案例

## 1.1.需求说明 

把你的生日字符串(格式：1992-10-20)转换为对应的日期对象. 

## 1.2. 步骤描述

  1. 定义类，提供main方法,在main方法中
  2. 创建SimpleDateFormat对象 df并指定日期格式
  3. 调用df的parse(String str)方法,传入你的生日字符串; 接收返回的日期对象 
  4. 打印这个日期对象
